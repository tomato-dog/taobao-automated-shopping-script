# taobao-automated-shopping-script

写在前面：

淘宝网页版抢购，自动选择并结算提交，⚠提交界面可能被检测到脚本提示系统繁忙，建议自己手动提交。

使用说明：

1.购物车里只保留需要抢购的商品，脚本会点击全选再结算，需要在代码里修改抢购时间，扫码登录成功后在终端里按回车键继续，提示手动抓元素后再按回车键，然后脚本就进入抢购程序了。

2.需安装msedgedriver，代码里的Edge路径需自行修改，用的是Microsoft Edge。

3.网页版淘宝会检测脚本，如果提交订单后提示【对不起，系统繁忙，请稍候再试】说明被屏蔽了，建议等系统自动结算后手动提交订单。

4.代码里已抓取全选/结算/提交订单的XPATH，如果后续页面有改动，请按代码里的步骤自行修改。
